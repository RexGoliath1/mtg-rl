Mechanics System¶

The mechanics system decomposes MTG cards into primitive mechanics (not text embeddings).

Vocabulary (1387 primitives)¶

MTG Mechanics Vocabulary

A comprehensive vocabulary of ~150+ mechanics primitives that can represent any Magic: The Gathering card. This is the foundation for the AlphaZero-style card encoding system.

Design principles: 1. Primitives are ATOMIC - they represent single mechanical concepts 2. Cards are SEQUENCES of primitives 3. New mechanics = new combinations of existing primitives (usually) 4. When truly new, we add minimal new primitives

The network learns how primitives INTERACT through self-play, not pre-coding.

Card Parser¶

Converts Oracle text to mechanics sequences.

Card Text Parser

Parses Scryfall oracle text into mechanics sequences.

Strategy: 1. Rule-based parsing for common patterns (handles ~80% of cards) 2. LLM fallback for complex/ambiguous text (handles remaining ~20%)

This is NOT meant to be exhaustive from day 1. We iterate: - Start with common patterns - Add rules as we encounter new patterns - LLM handles edge cases during development

src.mechanics.card_parser.strip_reminder_text(text: str) → str[source]¶

Strip parenthetical reminder text from oracle text.

Reminder text in MTG cards appears in parentheses and explains mechanics but doesn’t represent parseable abilities. Stripping it before word counting prevents artificial confidence deflation.

Example: “Flying (This creature can’t be blocked except by creatures with flying or reach.)” -> “Flying “

Precompute Embeddings¶

HDF5 pre-computation for all cards.

Pre-compute Card Mechanics Embeddings

Fetches all MTG cards from Scryfall and pre-computes mechanics encodings, storing them in HDF5 format for fast runtime access.

Output: data/card_mechanics.h5 - mechanics: (num_cards, vocab_size) binary matrix - parameters: (num_cards, max_params) float matrix - card_index: JSON mapping card_name -> row index - metadata: vocab_size, format, timestamp

Usage:

python -m src.mechanics.precompute_embeddings –format commander python -m src.mechanics.precompute_embeddings –format standard python -m src.mechanics.precompute_embeddings –format all
