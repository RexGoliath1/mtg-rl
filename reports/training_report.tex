\section{Bug Fixes Applied}

\begin{enumerate}
\item \textbf{PlayerControllerExternal.playChosenSpellAbility()}:
      Was returning \texttt{true} without executing.
      Fixed to call \texttt{ComputerUtil.handlePlayingSpellAbility()} for spells
      and \texttt{sa.resolve()} for lands.
\item \textbf{Land ability detection}: Was using \texttt{getFirstSpellAbility()}
      which returns mana abilities. Fixed to iterate \texttt{getSpellAbilities()}
      and check \texttt{isLandAbility()}.
\end{enumerate}
